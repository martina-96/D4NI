D4NI è un servizio web che sfrutta un algoritmo intelligente che ha lo scopo di creare dati strutturati per corpus.
Permette di mostrare diversi tipi di ooutput che un'analisi testuale può fornire ed effettuare il download dei dati nella forma più comoda per essere usati con Spektral.
È possibile effettuare il download del codice su github e modificarlo a proprio piacimento.
